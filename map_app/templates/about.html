About the Autonomous Car Project

Overview

The Autonomous Car project is an innovative IoT-based initiative designed to create a small-scale autonomous car that navigates between predefined points while detecting and avoiding obstacles. The goal is to ensure a smooth and efficient journey through advanced pathfinding and real-time obstacle detection using ultrasonic and infrared sensors.

Technology Stack

  • Hardware: GPS module (Neo6M V2 u-Blox), Raspberry Pi, L298N motor drivers, ultrasonic and infrared sensors.
  • Software: Python, OpenCV, IoT cloud services for remote monitoring and data management.
  • Connectivity: Real-time video streaming and sensor data via cloud integration.
